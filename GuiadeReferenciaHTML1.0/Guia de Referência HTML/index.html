IMG Insere uma imagem
Atributo Descrição
SRC (Obrigatório) Endereço completo da imagem, com caminho e extensão.

Pode ser informado com valores exatos como "200" ou proporcionais, como "50%".
HEIGHT Largura da imagem.
Pode ser informado com valores exatos como "200" ou proporcionais, como "50%".
Informando ambas Largura e Altura, a imagem terá a dimensão exata informada, informando apenas uma das duas a imagem mudará de tamanho proporcionalmente (recomendado).
ALT Texto alternativo caso a imagem não seja exibida.
Também utilizado para acessibilidade.

Valores: left | right
TITLE Texto exibido após alguns segundos caso o mouse seja posicionado sobre este elemento.

HEADER Tag semântica para cabeçahos

Utilizado para o cabeçalho do site ou de uma notícia, dentro de um ARTICLE, por exemplo, onde o HEADER poderia conter o título da notícia e sua data de publicação.

FOOTER Tag semântica para rodapés

Utilizado para o rodapé do site ou de uma notícia, dentro de um ARTICLE, por exemplo, onde o FOOTER poderia conter as tags da notícia e uma lista de notícias semelhantes.

NAV Tag semântica para áreas de navegação

Utilizado para inserir o menu do site ou qualquer outro tipo de barra de navegação.

SECTION Tag semântica para divisões visuais no site

Utilizado para definir uma área do site que esteja separado de outra, por exemplo, poderia se criar uma SECTION em uma galeria de imagens onde seria exibida as miniaturas das fotos e uma outra SECTION para a área onde a foto maximizada seria exibida.

ARTICLE Tag semântica para conteúdo do site

Utilizado para conter conteúdo auto-suficiente, ou seja, uma área de conteúdo que faça sentido se mostrada fora do site.

Imagine que você possui um blog com 10 postagens por página, cada postagem deve ser um ARTICLE, pois se você compartilhar somente o conteúdo da tag ARTICLE você estará compartilhando a postagem toda, sem precisar do blog para que a postagem faça sentido.

Um ARTICLE também pode conter tags HEADER, FOOTER e SECTION, por exemplo, dentro dela.

ASIDE Tag semântica para barras laterais com conteúdo genérico

Utilizado para criar uma área com propagandas, links para outros sites, links para matérias do próprio site, promoções, etc.

 

Atributos seletores

ID Seletor único

O atributo ID é utilizado para identificar um elemento de forma única, ou seja, nunca deve haver mais de um elemento com o mesmo ID. Seria semelhante à chave primária, no Banco de Dados, ou ao CPF das pessoas.

Utilizamos o ID quando precisamos pegar um elemento em específico, um botão específico que realizaá uma ação quando clicado, por exemplo.

<button id="imprimir">Clique para imprimir</button>
<button id="zoom">Clique para aumentar</button>
CLASS Seletor de grupo

O atributo CLASS é utilizado para identificar que o elemento pertence a um grupo e se comportará da mesma forma que outros membros do grupo.

Imagine que você quer que todo link que aponte para um produto Masculino possua a fonte Azul e todo link pqra um profuto Feminino possua a fonte Rosa, você pode definir classes que vão permitir que seus links pertençam a seus grupos específicos.

PS. Lembre-se que as classes informadas no CLASS serão contruídas em CSS.

/* Código CSS */
.feminino {
   color: #FF66CC;
}
.masculino {
   color: #3399FF;
}
